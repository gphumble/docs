The **Ticketing** tab is where you define what people can buy or reserve. At least one active tier is required before you can [publish](/events/publish-an-event).

## Ticket tiers

A **tier** is a buying option—General Admission, VIP, Early Bird, a sponsor table, and so on. Add as many as you need. For each tier you can set:

| Field | Purpose |
| --- | --- |
| **Name** | What buyers see (e.g., "VIP") |
| **Description** | Optional detail about what's included |
| **Price** | The cost (paid events) |
| **Capacity** | Optional limit on how many can be sold |
| **Admissions per ticket** | How many people one ticket admits |
| **Visibility** | Public or hidden |

<Note>
  For a **free** event, tiers still define the RSVP options—there's just no price.
</Note>

## Merchandise

If you turned on **Sell merch** when creating the event, a merch section appears here. Add items with a **name**, **price**, and optional **inventory** limit—they show up as optional add-ons in the checkout cart. Leave inventory blank for unlimited stock; otherwise each sale counts down and selling stops at zero.

<Tip>
  Use a hidden tier for comps or sponsor seats you don't want listed publicly, then add those guests yourself from the [Registrants](/events/registrants) tab.
</Tip>
