The **Contacts** page lists every household, organization, and foundation in your account. Use search and filters to narrow the list, then click any name to open that contact's profile.

## Search

Start typing in the search box at the top of the list. Advocora matches on:

- Contact **name**
- **Email** address
- **City**
- **ZIP / postal code**

Results update as you type, so you usually only need a few characters.

## Filters

Narrow the list using the filter dropdowns:

| Filter | What it does |
| --- | --- |
| **Donor status** | Show only Active donors, Inactive donors, or Non-donors. [Learn how status is calculated.](/contacts/donor-status) |
| **Type** | Show only Households, Organizations, or Foundations. |
| **State** | Show contacts in a specific U.S. state. |
| **County** | Show contacts in a specific county. |

<Tip>
  Filters stack. For example, combine **Active donor** \+ **State: Texas** to see only your active donors in Texas.
</Tip>

## What the columns show

The list displays each contact's **Name**, **City**, **County**, **State**, **Donor status**, and **Type**. Donor status and type appear as color-coded badges so you can scan the list quickly.

## Paging through results

At the bottom of the list you can:

- Choose how many contacts to show per page (**10, 25, 50, or 100**).
- Move between pages with the page controls.

When you change a filter, the list jumps back to the first page so you never see stale results.

## Common tasks

<AccordionGroup>
  <Accordion title="See only my active donors">
    Set the **Donor status** filter to **Active donor**.
  </Accordion>

  <Accordion title="Find one specific person">
    Type their name or email into the search box.
  </Accordion>

  <Accordion title="See everyone in one county">
    Use the **County** filter (optionally combined with **State**).
  </Accordion>
</AccordionGroup>
