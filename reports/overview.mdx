**Reports** turn your data into numbers you can act on—and share. Every report reads on screen with charts and tables, exports to a tidy PDF for the board or a clean CSV for the spreadsheet, and can be scheduled to arrive in your inbox.

<Note>
  Reports differ from [Queries](/queries/build-a-query). A query is an ad-hoc list you build; a report is a polished, pre-built view with charts, comparisons, and delivery options.
</Note>

## The report library

The library groups reports by category. **Pin** the ones you read often—pinned reports rise to the top.

| Category | Covers |
| --- | --- |
| **Giving** | Contributions, designations, accounts, top donors |
| **Recurring** | Pledges, subscriptions, churn and drift |
| **Memberships** | Tiers, renewals, lifetime value |
| **Events** | Performance, profit & loss, fundraising |
| **People** | Acquisition, retention, donor health |
| **Acquisition** | Forms, sources, attribution |
| **Geography** | State, congressional district |
| **Engagement** | Email signals, score tiers |
| **Operations** | Reconciliation, refunds |

<Note>
  Membership reports appear only when the Memberships module is enabled.
</Note>
